h1. Monitoring the application post deployment using the Cloudify client

Aside from the viewing the current status of the your deployed applications on Cloudify's WEB UI, 
you can also do it with the Cloudify console.


In order to see which application are deployed and running, type:
list application

In order to see which services are deployed and running, type:
list service

To get a list of all instances of a certain service and their correspoding hosts, type:
list-instances serviceName
