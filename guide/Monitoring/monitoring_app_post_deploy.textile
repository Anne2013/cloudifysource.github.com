h1. Monitoring the application post deployment using the Cloudify client

h4. Aside from the viewing the current status of the your deployed applications on Cloudify's WEB UI, 
you can also do it with the Cloudify console.

h4. In order to see which applications are deployed and running, type:
<script src="https://gist.github.com/1723921.js?file=listApplication.bat"></script>

h4. In order to see which services are deployed and running, type:
<script src="https://gist.github.com/1723925.js?file=listService.sh"></script>

h4. To get a list of all instances of a certain service and their correspoding hosts, type:
<script src="https://gist.github.com/1723927.js?file=listInstances.groovy"></script>
