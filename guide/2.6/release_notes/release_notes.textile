h1. Overview

h4. Below is a list of new features, improvements, fixed issues, and known issues in Cloudify.
We track all the issues in our public "JIRA":https://cloudifysource.atlassian.net.


*Cloudify 2.6 GA*  - published on July 11, 2013

<hr/>

"Cloudify 2.6 RC":#v26rc  - published on Jun 27, 2013
This version contains the following key improvements and cool features:
*  "(newwindow)Rest API":../restapi/restdoclet.html - Cloudify now has an enhanced Rest API.
*  "IP addresses filters":../clouddrivers/cloud_driver#ipfilter - Users are now able to configure Private and Public IP Addresses filters.
*  "Bootstrapping Validations":../bootstrapping/bootstrapping_validations - Bootstrapping validations have been added to help avoid a bootstrap failure.
*  "Recipe Repository Enhancement":../setup/recipes_tab - Users can now track installations progress from the Web UI.

<br/>
<hr/>

"Cloudify 2.6 M4":#v26m4  - published on Jun 12, 2013
This version contains the following key improvements and cool features:
*  "Bootstrapping Validations":../bootstrapping/bootstrapping_validations - Bootstrapping validations have been added to help avoid a bootstrap failure.
*  "Recipe Repository Enhancement":../setup/recipes_tab - Users can now track installations progress from the Web UI.
